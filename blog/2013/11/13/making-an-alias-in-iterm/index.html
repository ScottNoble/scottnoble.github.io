Making an Alias in iTerm

Nov 13th, 2013

In class we use iTerm2. To learn and get familiar with the command line we completed all the exercises in http://cli.learncodethehardway.org/book/ and in class we do most of our work from the command line.

I finally decided that I no longer wanted to go through the hassle of “cd’ing” into my different class directories so I made some aliases so that I can go directly into the directories that I want.

Typing i goes into the VIM INSERT mode. I then created my aliases by typing alias name=‘cd /path’ Next, to exit the VIM INSERT I pressed ESC and then typed :wq to save and quit. My aliases looked like this:

Now, when the instructor tells us to go to our homework directory, the building the toolbelt directory or the workspace directory I just type hw, btt, or ws into the command line and I am there no matter what directory I was in to start.

To activate the aliases you need to exit your current session and reload the iterm.
